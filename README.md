## Installation

PickerCountry is available through [CocoaPods](http://cocoapods.org). PickerCountry for iOS supports iOS 8.x, iOS 9.x and iOS 10. To install
it, simply add the following line to your Podfile:

```ruby
pod 'PickerCountry'
```
